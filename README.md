## 🚀 快速开始

本站使用 [Docsify](https://docsify.js.org/) 构建，支持在线阅读和本地预览。

### 在线访问

直接访问 GitHub Pages：[https://clowniii.github.io](https://clowniii.github.io)

### 本地预览

```bash
# 安装 docsify-cli
npm i docsify-cli -g

# 运行本地服务器
docsify serve .

# 访问 http://localhost:3000
```
